Full-Waveform Inversion for Time-Harmonic Elastography using Ultrasound Shear Waves

University of Oslo ‚Äî Ultrasound Elastography Project
DScience/DSTrain: Postdoctoral Fellow Profile

This research is part of the Ultrasound Shear Wave Elastography Project at DSB group, which aims to advance medical ultrasound elastography for estimating tissue mechanical properties‚Äîessential for tumor detection and disease staging. Our group is particularly interested in developing methods for ultrasound shear-wave elastography using external time-harmonic excitations, with the goal of creating 3D myocardial elasticity maps of the heart.

My specific project focuses on integrating Full-Waveform Inversion into the processing of shear-waves by incorporating the physics of wave propagation.

Seismic imaging of dam-rock foundation interface

This project started during my previous postdoctoral position at IDEFIX team. It consists in a collaborative work between INRIA and EDF R&D. The project addresses interface imaging problems in electricity production sites. The primary case study focuses on imaging the dam‚Äìrock foundation interface, using non-destructive seismic waves. This interface is a critical factor for ensuring structural stability.

Our approach combines shape optimization techniques with the Full-Waveform Inversion framework to obtain high-resolution images of the interface from seismic measurements acquired non-invasively on the dam wall. Currently, we are working on expanding this methodologies in 3D and improving the forward model to make it applicable in operational conditions.
